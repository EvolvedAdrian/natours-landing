# 🍃 Natours - The Best Tours
![Version](https://img.shields.io/badge/version-1.0.0-blue.svg?cacheSeconds=2592000)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](#)

> The landing page for Natours, a tour agency offering a diverse range of guided adventures, including sea, forest, and snow expeditions designed for varying skill levels and experiences.

### Project Sections:
* **Header:** Logo, navitation and hero section, which includes main title and subtitle with background natoure image and CTA button.
* **About:** Some engaging copy to attract customers + Learn more link.
* **Features:** Outlines the benefits of booking a tour with us.
* **Tours:** View the range of tours + CTA button.
* **Testimonials:** Testimonial section is very important for building trust + Read all stories link.
* **CTA:** Form for users to book a tour.
* **Footer:** Company links

## 🍃 Installation & Requirements

### Prerequisites

To run this project locally, you must have **Node.js** installed. You can check your version by running `node -v` in your terminal.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/EvolvedAdrian/natours-landing.git
    ```
2.  **Navigate to the directory:**
    ```bash
    cd natours-landing
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```

---

## 💻 Usage

Start the application using:

```bash
npm run start
```
